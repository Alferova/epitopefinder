Analysis scriptsÂ¶

A variety of small scripts are included for analyzing and plotting the results of running the main epitopefinder scripts.

These scripts mostly require matplotlib, so you need to download and install this package in order to utilize the scripts. A few functions also use scipy.

  • epitopefinder_selectsites.py
  • epitopefinder_plotlineardensity.py
  • epitopefinder_plotcorrelation.py
  • epitopefinder_plotdistributioncomparison.py
